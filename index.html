Hmm Cavalry

Descrição :

Hmm Cavalry é um jogo na qual um cavaleiro quer incrementar seu dicionario com a lendaria palavra "Hmm", mas para isso ele tera que passar por desafios muito difíceis, como um espinho que perfura o chão atras dele, porém para ele sobreviver em cada lugar havera uma ajuda que nosso guerreiro tera de se aproveitar delas.

Imagens :



Versão :

Beta

Avisos :

O Jogo contem um sistema de save, para ele ser ativado o jogo precisa de permissão ao armazenamento, o app não fara isso automaticamente, por tanto vá manualmente as configurações do game e conceda as permissões. Como é uma beta o jogo não esta completo, então não espere um game grande, isso é uma beta privada, o download só sera desbloquedo se você colocar a senha corretamente.
